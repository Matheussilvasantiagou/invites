# Convite de Aniversário (GitHub Pages)

Este repositório publica um convite digital via **GitHub Pages**.

## Como publicar

1. Faça commit e envie para o GitHub (branch `main`).
2. No GitHub, abra **Settings → Pages**.
3. Em **Build and deployment**:
   - **Source**: *Deploy from a branch*
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. Salve e aguarde o link do Pages aparecer.

## Como editar o convite

Edite o arquivo `index.html` e troque textos como:
- Nome (“ADRIANA'S 25/04”)
- Data e horário
- Texto do convite
- Endereço / mapa / RSVP

## Estrutura

- `index.html`: página do convite (estático).
